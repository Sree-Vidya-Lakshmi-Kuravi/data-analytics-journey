## Dataset
Superstore sales dataset containing order, customer, product, and sales transaction details across multiple years.

### Data Preparation
- Data cleaned and transformed using Power Query
- Removed nulls and corrected data types
- Renamed columns for clarity and consistency
### Data Model
- Single fact table containing sales transactions
- Clean and simple model design
- Optimized for reporting and analysis without unnecessary complexity
### KPIs Used
- Total Sales
- Total Customers
- Average Sales per Order
### Visuals Included
- Sales by Category
- Sales by Segment
- Sales Trend by Year
- Sales by Sub-Category
### Key Insights
- Technology category generates the highest revenue
- Consumer segment dominates overall sales contribution
- Business shows steady year-over-year growth
- Certain sub-categories contribute low sales despite high product counts

## Recommendation
- Focus on expanding high-performing categories such as Technology, while reviewing pricing or strategy for lower-performing sub-categories to improve overall profitability.
